Blazor: wasm en C#

Blazor es la propuesta de Microsoft para trabajar en aplicaciones web utilizando C# y tambien es su propio framwork de frontend

Los requisitos necesarios es tener instalado .NET con las opciones de "ASP.NET y web development" instaladas. (Es recomendable tener visual-studio instalado ya que desde ahi se pueden descargar todo lo necesario)



Una vez teniendo todo lo necesario hay que crear una aplicacion de blazor, ya sea desde consola con el comando: dotnet new blazorwasm -o AppName
o a travez de las opciones de visual-studio

Una vez teniendo el proyecto se creara una pagina dummy que por defecto ya tiene la coneccion para correr la aplicacion.
Dentro de la carpeta se encontrara un subfolder llamado pages. AHi se encuentran los archivos .razor que son los componentes de la app.
Dentro de cada componente hay una etiqueta @page que define la ruta de la pagina, tags de html y una etiqueta @code donde va el codigo de C#

Ademas dentro de la carpeta shared se encuentra el archivo NavMenu que contiene la lista de las paginas que se mostraran

Para correr la aplicacion se pude usar el comando run o correr la aplicacion desde visual-studio
